Gonfigurações globais da CSS3

Para customizar o fundo da tela tem que criar o codigo como esta abaixo.

  •                 
                    body {
                        height: 100%;
                        background-image: linear-gradient(to bottom, #3198E2, #6D59C0, #B93590, #FDD579);
                        background-attachment: fixed;
                    }

                    body {
                        height: 100%;
